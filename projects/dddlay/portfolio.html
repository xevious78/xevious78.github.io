Dddlay

Audio Plugin for Newbies. (2014 - 2015)

Dddlay is a fun-to-use audio plugin made specifically to help new audio producers create the perfect echo effect in an easy way.


I first designed the User Interface using Sketch. Then, I developped the front-end in C++ using the WDL/IPlug framework while my teammate focused on the Audio Processing algorithms.
