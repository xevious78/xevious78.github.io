tidy

Organize Everything ! (2014)

tidy is a school project that aims to reinvent the "trinklet bowl". This device allows the user to create dedicated spaces that holds specific objects. tidy is then able to reminds the user to take these objects before leaving.


My teammate worked on the physical design whereas worked on the User Interface. I used Adobe After Effects to make different mockups and then took advantage of Cocoa/Objective-C to build a prototype of the User Interface.
